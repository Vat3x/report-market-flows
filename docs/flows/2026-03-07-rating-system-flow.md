         ▼
    ┌──────────────────┐
    │ ფილტრი:          │
    │ status == VERIFIED│
    │ only             │
    └────────┬─────────┘
             │
             ▼
    ┌──────────────────────────────────┐
    │ თითოეულ რეპორტს → severity weight│
    │                                  │
    │ ტრანსპორტი:                      │
    │   LATE_DELIVERY      →  5       │
    │   CARGO_DAMAGE       → 10       │
    │   ACCIDENT           → 15       │
    │   DOT_INSPECTION     →  8       │
    │                                  │

    │   DOCUMENTATION      →  5       │
    │   IRRESPONSIBILITY   →  8       │
    └────────┬─────────────────────────┘
             │
             ▼
    ┌──────────────────────────┐
    │ totalDeduction = Σ weights│
    └────────┬─────────────────┘
             │
             ▼
    ┌──────────────────────────┐
    │ score = max(0, 100 -     │
    │              totalDeduction)│
    └────────┬─────────────────┘
             │

    ┌──────────────────────────┐
    │ ფერის განსაზღვრა:        │

## მაგალითები

### მაგალითი 1: ახალი მძღოლი, 2 რეპორტი

```
რეპორტები:
  ├─ LATE_DELIVERY (VERIFIED)    → -5
  └─ COMMUNICATION (VERIFIED)    → -3

გამოთვლა:
  100 - (5 + 3) = 92%

შედეგი:
  ┌────────────────────────────────────┐
  │          92%  🟢 Good Standing     │
  │  ████████████████████████████░░░░  │
  │                                    │
  │  რეპორტები: 2                     │
  │  ჩამოჭრა: -8 ქულა               │
  └────────────────────────────────────┘
```

### მაგალითი 2: პრობლემური მძღოლი, 5 რეპორტი

```
რეპორტები:
  ├─ ACCIDENT (VERIFIED)           → -15
  ├─ CARGO_DAMAGE (VERIFIED)       → -10
  ├─ DOT_INSPECTION (VERIFIED)     → -8
  ├─ PUNCTUALITY (VERIFIED)        → -4
  └─ LATE_DELIVERY (DISPUTED)      → არ ითვლება

გამოთვლა:
  VERIFIED მხოლოდ: 15 + 10 + 8 + 4 = 37
  100 - 37 = 63%

შედეგი:
  ┌────────────────────────────────────┐
  │          63%  🟡 Needs Attention   │
  │  ████████████████████░░░░░░░░░░░  │
  │                                    │
  │  რეპორტები: 5 (1 disputed)       │
  │  ჩამოჭრა: -37 ქულა              │
  └────────────────────────────────────┘
```

### მაგალითი 3: კრიტიკული მძღოლი

```
რეპორტები:
  ├─ ACCIDENT (VERIFIED)           → -15
  ├─ ACCIDENT (VERIFIED)           → -15
  ├─ CARGO_DAMAGE (VERIFIED)       → -10
  ├─ IRRESPONSIBILITY (VERIFIED)   → -8
  ├─ DOT_INSPECTION (VERIFIED)     → -8
  └─ PUNCTUALITY (VERIFIED)        → -4

გამოთვლა:
  15 + 15 + 10 + 8 + 8 + 4 = 60
  100 - 60 = 40%

შედეგი:
  ┌────────────────────────────────────┐
  │          40%  🔴 Critical          │
  │  ████████████░░░░░░░░░░░░░░░░░░░  │
  │                                    │
  │  რეპორტები: 6                     │
  │  ჩამოჭრა: -60 ქულა              │
  └────────────────────────────────────┘
```

### მაგალითი 4: ნულოვანი რეიტინგი

## Dispute-ის გავლენა რეიტინგზე

```
  მდგომარეობა 1: Dispute-მდე
  ─────────────────────────────
  რეპორტები (VERIFIED):
    ACCIDENT: -15
    CARGO_DAMAGE: -10
    COMMUNICATION: -3

  რეიტინგი = 100 - 28 = 72% 🟡


  მდგომარეობა 2: CARGO_DAMAGE გასაჩივრებულია
  ─────────────────────────────────────────
  რეპორტები (VERIFIED):
    ACCIDENT: -15
    COMMUNICATION: -3
  რეპორტები (DISPUTED):
    CARGO_DAMAGE: (არ ითვლება)

  რეიტინგი = 100 - 18 = 82% 🟢

  ცვლილება: 72% → 82% (+10 ქულა)
```

## Edge Cases

| სიტუაცია | შედეგი |
|---------|--------|
| არცერთი რეპორტი | 100% 🟢 "სუფთა ჩანაწერი" |
| მძღოლი DB-ში არ არის | "ჩანაწერი ვერ მოიძებნა — 100%" |
| ქულა < 0 | max(0, score) → 0% |
| ყველა PENDING | 100% + შენიშვნა: "X რეპორტი განხილვაშია" |
| ყველა DISPUTED | 100% + შენიშვნა: "X რეპორტი გასაჩივრებულია" |
| შერეული სტატუსები | მხოლოდ VERIFIED ითვლება, დანარჩენი ცალკე ჩანს |
