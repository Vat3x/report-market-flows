# Report Market - პროექტის მოკლე აღწერა და ხარჯები

## პროექტის მიმოხილვა
**Report Market** — ამერიკის სატვირთო გადაზიდვების (Trucking) ინდუსტრიის მძღოლების რეპუტაციის პლატფორმა.
**პრობლემა**: ქერიერებსა და დისპეტჩერებს არ აქვთ ცენტრალიზებული სისტემა მძღოლების საიმედოობის შესამოწმებლად.
**გადაწყვეტა**: ნეგატიური რეპორტების სისტემა — მძღოლი იწყებს 100%-იანი რეიტინგით, ყოველი ვერიფიცირებული რეპორტი ამცირებს ქულას.

## ხარჯების ანალიზი
*   **Development ეტაპი (უფასო):** ~$1/თვე (მხოლოდ დომენი)
*   **პროდაქშენი (რეკომენდირებული):** ~$46/თვე 
    *   Vercel Pro ($20)
    *   Supabase Pro ($25)
    *   დომენი (~$1)
*   **ალტერნატივა (Neon-ით):** ~$40/თვე
